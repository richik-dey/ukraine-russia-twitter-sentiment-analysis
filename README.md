Ukraine vs Russia Conflict – Twitter Sentiment Analysis

This project performs sentiment analysis on Twitter data related to the Russia-Ukraine conflict using Natural Language Processing and Machine Learning techniques. It analyzes public opinion by classifying tweets into positive, negative, and neutral sentiments and provides insights into global discourse around the conflict.

Project Overview

Domain: Natural Language Processing (NLP), Social Media Analytics
Techniques: VADER Sentiment Analysis, TF-IDF Vectorization, Text Preprocessing, Machine Learning Classification
Algorithms Used: Logistic Regression, Naive Bayes, Support Vector Machine, Decision Tree, Random Forest, K-Nearest Neighbor
Dataset Size: 47,994 Tweets
Best Accuracy: Linear Support Vector Machine – 78.48%

Workflow

Collect Twitter dataset related to Russia-Ukraine conflict
Perform language detection and filter English tweets
Clean text (remove URLs, mentions, hashtags, punctuation)
Apply VADER sentiment scoring
Classify tweets into Positive, Negative, Neutral
Extract features using TF-IDF and Bag-of-Words
Split dataset into training and testing sets
Train six machine learning models
Evaluate using Accuracy, Confusion Matrix, ROC Curve
Visualize sentiment distribution and word clouds
Implement real-time sentiment prediction

Key Findings

Neutral Sentiment: 55.97%
Negative Sentiment: 25.11%
Positive Sentiment: 18.92%
Best Model: Linear SVM with 78.48% accuracy
